# INFARED

![logo](https://i.postimg.cc/BbYDZRn1/image.png)

INFARED is a guide to set up the **Ultraviolet web proxy** on [Vercel](https://vercel.com/). It provides step-by-step instructions to deploy a lightweight, secure, and fast web proxy, enabling you to access content safely and efficiently.

This is meant for minimal projects, but can be used effectively in production.

---

## Features

- Easy deployment on Vercel
- Lightweight and fast (uses vercel's servers)
- No configuration required

---

## Requirements

- A **Vercel account:** ([sign up here](https://vercel.com/signup))
- Basic understanding of GitHub and Vercel deployments
- Node.js installed (for local testing, optional)
